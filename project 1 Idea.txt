ppmrw function
	one convert functions
	if first argument is 3 ---> convert to P3
	if first argument is 6 ---> convert to P6
	
Read image in and put into data structure of some sort

To convert to P3:
	Change P# to P3
	Skip over comments
	Skip over picture dimensions
	convert each set of characters to ascii
	save to output.ppm
	export image into directory

To convert to P6:
	Change P# to P6
	Skip over comments
	Skip over picture dimensions
	convert each set of characters to rawbits
	save to output.ppm
	export image into directory
